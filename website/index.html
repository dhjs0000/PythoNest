关于 PythoNest

PythoNest 是一个基于 PyQt6 开发的开源 Python 版本管理工具，旨在提供简单、直观的界面帮助用户管理多个 Python 环境。

项目遵循 GNU 通用公共许可证第3版(GPL-3.0)，您可以自由使用、修改和分发本软件。
